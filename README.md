### インストール手順
1. 夜桜.col／薄藤.col をダウンロード
2. サクラエディタを開き、設定→タイプ別設定→カラー→インポート から1でダウンロードしたファイルを選択。

インポート後、細かい設定項目はご自由に変更してお使いください🌸
